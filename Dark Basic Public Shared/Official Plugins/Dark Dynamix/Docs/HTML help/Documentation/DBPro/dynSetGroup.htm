DYN SET GROUP (int actorID, int group)

This command allows you to set which collision group the given actor belongs to, all actors start off in group 0.

Parameters:
actorID - ID number of the object that the actor represents in the simulation.
group - Group number you wish to assign the given actor to.


Returns:
None

DYN SET GROUP (int objectID, int limbID, int group)

This command allows you to set which collision group the given actor belongs to, all actors start off in group 0.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
group - Group number you wish to assign the given actor to.


Returns:
None
