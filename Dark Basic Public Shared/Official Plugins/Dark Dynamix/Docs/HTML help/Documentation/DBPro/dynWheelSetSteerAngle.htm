DYN WHEEL SET STEER ANGLE (int objectID, int shapeIndex, float angle)

Use this command to set the steer angle of the given wheel shape.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
angle - Value you wish to set the steer angle to, in degrees.


Returns:
None
