DYN JOINT DESC SET LOCAL AXIS (int jointDescID, int AorB, float x, float y, float z)

Sets the local axis of one of the two actors associated with the joint descriptor. Most of the time you should use the 'global axis' command which sets the both actors local axis to the same frame in the world. The axis values should be taken from a normalised vector.

Parameters:
jointDescID ID number of the descriptor.
AorB - 0 for actorA, 1 for actorB.
x - Axis about the x axis in the space of the given actor.
y - Axis about the y axis in the space of the given actor.
z - Axis about the z axis in the space of the given actor.


Returns:
None
