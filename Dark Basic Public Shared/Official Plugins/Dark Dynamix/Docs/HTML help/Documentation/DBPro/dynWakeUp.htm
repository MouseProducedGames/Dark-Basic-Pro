DYN WAKE UP (int objectID, float wakeCounterValue)

Wakes up the given actor for a given amount of time.

Parameters:
objectID - ID number of object the actor represents in the simulation.
wakeCounterValue - Amount of time the actor should remain awake for, a value of 0 is equivalent to putting the actor to sleep.


Returns:
None


-------------------------------------------------------------------------------
DYN WAKE UP (int objectID, int limbID, float wakeCounterValue)

Wakes up the given actor for a given amount of time. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of limb the actor represents in the simulation.
wakeCounterValue - Amount of time the actor should remain awake for, a value of 0 is equivalent to putting the actor to sleep.


Returns:
None
