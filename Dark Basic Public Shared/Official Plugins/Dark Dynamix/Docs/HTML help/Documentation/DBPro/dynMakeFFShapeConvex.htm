bool DYN MAKE FF SHAPE CONVEX (int posVector3ID, int rotVector3ID, int convexMeshID)

This command will add a convex shape to an internal force field shape list, when you create a force field shape group it uses the shapes contained in this list, the list can then be cleared.

Parameters:
posVec3ID - ID number of the vector containing position data.
rotVec3ID - ID number of the vector containing rotation data.
convexMeshID - ID number of the convex mesh which must have already been created or loaded from file.


Returns:
TRUE if succeeded, FALSE otherwise.
