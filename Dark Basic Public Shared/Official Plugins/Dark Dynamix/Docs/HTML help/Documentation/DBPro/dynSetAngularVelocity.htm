DYN SET ANGULAR VELOCITY (int objectID, float x, float y, float z)

Sets the angular velocity of an actor, you should usually apply forces to change the velocity of an actor although this command can be used to reset your actor etc.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
x - Velocity about the x axis.
y - Velocity about the y axis.
z - Velocity about the z axis.


Returns:
None

DYN SET ANGULAR VELOCITY (int objectID, int limbID, float x, float y, float z)

Sets the angular velocity of an actor, you should usually apply forces to change the velocity of an actor although this command can be used to reset your actor etc.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
x - Velocity about the x axis.
y - Velocity about the y axis.
z - Velocity about the z axis.


Returns:
None
