DYN KINEMATIC MOVE ROT (int objectID, float rotX, float rotY, float rotZ)

If an actor has been made kinematic then you may use this command to rotate the actor.

Parameters:
objectID - ID number of the object/actor.
rotX - Global orientation about the x axis.
royY - Global orientation about the y axis.
rotZ - Global orientation about the z axis.


Returns:
None
