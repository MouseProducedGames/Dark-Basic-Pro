DYN FLUID SET RANDOM ANGLE (int fluidID, int emitterIndex, float angle)

This command allows you to randomize the angle the particles leave the emitter, the default is for the particles to leave the emitter at the exact angle the emitter is facing, which is not always desirable.

Parameters:
fluidID - ID number of the fluid.
emitterIndex - Index of the emitter for the given fluid.
angle - Value in radians.


Returns:
None
