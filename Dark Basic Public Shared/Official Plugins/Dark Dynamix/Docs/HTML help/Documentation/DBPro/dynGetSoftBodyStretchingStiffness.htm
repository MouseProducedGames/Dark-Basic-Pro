float DYN GET SOFT BODY STRETCHING STIFFNESS (int softBodyID)

Gets the stretching stiffness of the given soft body. Volume and stretching stiffnesses are used the specify how strongly the soft body responds to deviations from the rest state given by the input mesh. The soft body simulation manipulates the vertex positions to counteract deformations that change the volume and edge lengths of each tetrahedral element.

Parameters:
softBodyID - ID number of the soft body.


Returns:
value - The current value of the stretching stiffness, the default value is 0.5.
