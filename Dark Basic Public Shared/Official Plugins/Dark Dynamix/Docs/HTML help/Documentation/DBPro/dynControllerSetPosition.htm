DYN CONTROLLER SET POSITION (int controllerID, float x, float y, float z)

Sets the position of the character controller, does not check for collisions. Use this to reset/teleport your character.

Parameters:
controllerID - ID number of the object the controller represents in the simulation.
x - New position along the x axis.
y - New position along the y axis.
z - New position along the z axis.


Returns:
None
