bool DYN MAKE CLOTH (int objectID, int clothMeshID)

Creates a cloth from a cloth mesh.

Parameters:
objectID - ID number of the object the actor will represent in the simulation.
clothMeshID - ID number of the mesh you wish to create the actor from.


Returns:
TRUE if succeeded, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN MAKE CLOTH (int objectID, int clothMeshID, unsigned int flags)

Creates a cloth from a cloth mesh.

Parameters:
objectID - ID number of the object the actor will represent in the simulation.
clothMeshID - ID number of the mesh you wish to create the actor from.
flags - Flags for the cloth, allows you to set certain properties for the cloth. If simulating on hardware or simulating with tearing you must set the flag here, all other flags can be set after creation if desired.


Returns:
TRUE if succeeded, FALSE otherwise.
