DYN GROUP PAIR SET FLAGS (int groupA, int groupB, int flag)

Sets the flag for the given pair of groups of actors, you can use this flag to enable contact reports between the two given actor groups.

Parameters:
groupA - First group.
groupB - Second group.
flag - Contact pair flag, this determines whether you wish to receive contacts on touch, start touch or end touch, or a combination.


Returns:
None
