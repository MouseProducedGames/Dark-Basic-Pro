DYN TIRE FUNC DESC SET EXTREMUM (int ID, float extremumSlip, float extremumValue)

This function will set the extremum variables of the given tire force function descriptor.

Parameters:
ID - ID number of the tire function descriptor.
extremumSlip - Extremal point of curve. Both values must be positive. Range:(0,inf) Default: 1.0
extremumValue - Extremal point of curve. Both values must be positive. Range:(0,inf) Default: 0.02


Returns:
None
