bool DYN LOAD CONVEX MESH (char* file, int convexMeshID)

Loads a convex mesh which has been pre-cooked to file.

Parameters:
file - Name of the mesh file you wish to load the mesh from.
convexMeshID - ID number you wish to assign to the convex mesh, use this number when creating any actor(s) that want to use this mesh.


Returns:
TRUE if succeeded, FALSE otherwise.
