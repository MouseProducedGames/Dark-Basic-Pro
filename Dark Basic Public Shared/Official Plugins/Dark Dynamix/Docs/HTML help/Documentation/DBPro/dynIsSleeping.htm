bool DYN IS SLEEPING ( int objectID )

This command allows you to check if an actor is currently asleep. Actors fall asleep when they are no longer moving, this means they are taken out of simulation until a force is applied either directly or from contact with another actor.

Parameters:
objectID - ID number of the object the actor represents in the simulation.


Returns:
TRUE if sleeping, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN IS SLEEPING ( int objectID, int limbID )

This command allows you to check if an actor is currently asleep. Actors fall asleep when they are no longer moving, this means they are taken out of simulation until a force is applied either directly or from contact with another actor.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb which the actor represents in the simulation.


Returns:
TRUE if sleeping, FALSE otherwise.
