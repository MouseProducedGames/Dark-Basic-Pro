DYN JOINT DESC SET GLOBAL ANCHOR (int jointDescID, float posX, float posY, float posZ)

Sets the the position of the joint in global space, this is then internally converted to the local space of each associated actor, for this reason it is important to set the actors first before calling this command.

Parameters:
jointDescID ID number of the descriptor.
posX Position along the global x-axis.
posY Position along the global y-axis.
posZ Position along the global z-axis.


Returns:
None
