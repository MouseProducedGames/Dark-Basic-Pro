DYN SHAPE PAIR SET FLAGS (int actorA, int shapeIndexA, int actorB, int shapeIndexB, int flag)

Sets the pair flag for the given pair of actor shapes, you can't use this flag to enable contact reports between the two given shapes, this is only possible at actor level. You can use this command to disable collisions 'NX_IGNORE_PAIR' between two specific shapes.

Parameters:
actorA - First actor of actor pair.
shapeIndexA - Shape index for the first actor, default shape is 0.
actorB - Second actor of actor pair.
shapeIndexB - Shape index for the second actor, default shape is 0.
flag - NX_IGNORE_PAIR is the only non-zero value allowed as a shape pair flag. Passing zero stops ignoring the shape pair flag again.


Returns:
None
