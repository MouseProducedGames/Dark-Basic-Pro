bool DYN IS WHEEL (int objectID, int shapeIndex)

This command will tell you if the given shape in a given actor is a wheel shape, wheel shapes have alot of additional commands as well as the regular shape commands.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.


Returns:
TRUE if given shape is a wheel, FALSE otherwise.
