DYN SET ANGULAR DAMPING (int objectID, float value)

Sets the angular damping of an actor.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
value - Angular damping value.


Returns:
None


-------------------------------------------------------------------------------
DYN SET ANGULAR DAMPING (int objectID, int limbID, float value)

Sets the angular damping of an actor. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb/bone the actor represents in the simulation.
value - Angular damping value.


Returns:
None
