DYN WHEEL SET MOTOR TORQUE (int objectID, int shapeIndex, float torque)

Use this command to set the motor torque of the given wheel shape, this can be positive or negative depending on direction.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
torque - Value you wish to set the motor torque to.


Returns:
None

DYN WHEEL SET MOTOR TORQUE (int objectID, int limbID, int shapeIndex, float torque)

Use this command to set the motor torque of the given wheel shape, this can be positive or negative depending on direction.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
torque - Value you wish to set the motor torque to.


Returns:
None
