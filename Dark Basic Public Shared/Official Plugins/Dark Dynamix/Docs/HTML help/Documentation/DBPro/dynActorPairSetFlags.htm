DYN ACTOR PAIR SET FLAGS (int actorA, int actorB, int flag)

Sets the pair flag for the given pair of actors, you can use this flag to enable contact reports between the two given actors.

Parameters:
actorA - First actor of actor pair.
actorB - Second actor of actor pair.
flag - Contact pair flag, this determines whether you wish to recieve contacts on touch, start touch or end touch, or a combination.


Returns:
None
