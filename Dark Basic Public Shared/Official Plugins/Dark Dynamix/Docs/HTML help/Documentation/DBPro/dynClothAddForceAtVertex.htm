DYN CLOTH ADD FORCE AT VERTEX (int clothID, int vertexID, int forceVec3ID)

This command will apply the given force at the given cloth vertex.

Parameters:
clothID - ID number of the object the cloth represents in the simulation.
vertexID - ID number of the vertex you wish to apply the force to.
forceVec3ID - ID number of the vector containing the force to be added.


Returns:
None
