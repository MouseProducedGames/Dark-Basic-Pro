bool DYN WHEEL GET CONTACT (int objectID, int shapeIndex)

This command will tell you if the given wheel shape is currently in contact with anything. After using this command, if the wheel shape is currently in contact with anything you can use the other contact commands to get more information about the contact.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.


Returns:
TRUE if in contact, FALSE otherwise.
