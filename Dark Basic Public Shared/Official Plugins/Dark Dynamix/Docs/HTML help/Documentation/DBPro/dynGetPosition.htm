DYN GET POSITION ( int objectID, int outVec3ID )

This command calculates and provides you with the given actors position. You can also use the dark basic position commands.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
outVec3ID - ID number of the vector3 the result will be placed into.


Returns:
None

DYN GET POSITION ( int objectID, int limbID, int outVec3ID )

This command calculates and provides you with the given actors position. You can also use the dark basic position commands.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb which the actor represents in the simulation.
outVec3ID - ID number of the vector3 the result will be placed into.


Returns:
None
