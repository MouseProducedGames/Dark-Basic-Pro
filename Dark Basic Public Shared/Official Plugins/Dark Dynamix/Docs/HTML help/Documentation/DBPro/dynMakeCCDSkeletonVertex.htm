bool DYN MAKE CCD SKELETON VERTEX (int objectID, int limbID)

This will add a single vertex mesh to the given actor which will then be used for continuous collision detection if enabled.

Parameters:
objectID ID number of the object which the actor represents in the simulation.
limbID ID number of the limb which the actor represents in the simulation, if the actor was not created using a limb then set to 0.


Returns:
TRUE if succeeded, FALSE otherwise.
