DYN GET ANGULAR VELOCITY (int objectID, int vector3ID)

Fetches the angular velocity of an actor and places the values in the given vector.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
vector3ID - ID number of the vector you wish to use to collect the data.


Returns:
None
