DYN JOINT DESC SET LOCAL NORMAL (int jointDescID, int AorB, float x, float y, float z)

Sets the local normal of one of the two actors associated with the joint descriptor. The normal values should be taken from a normalised vector.

Parameters:
jointDescID ID number of the descriptor.
AorB - 0 for actorA, 1 for actorB.
x - Normal about the x axis in the space of the given actor.
y - Normal about the y axis in the space of the given actor.
z - Normal about the z axis in the space of the given actor.


Returns:
None
