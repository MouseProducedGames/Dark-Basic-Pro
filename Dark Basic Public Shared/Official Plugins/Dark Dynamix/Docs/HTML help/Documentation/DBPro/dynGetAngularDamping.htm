float DYN GET ANGULAR DAMPING (int objectID)

Returns the angular damping of an actor.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.


Returns:
Angular damping value.


-------------------------------------------------------------------------------
float DYN GET ANGULAR DAMPING (int objectID, int limbID)

Returns the angular damping of an actor. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb/bone the actor represents in the simulation.


Returns:
Angular damping value.
