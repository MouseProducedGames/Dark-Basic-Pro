DYN GET ORIENTATION ( int objectID, int outVec3ID )

This command calculates and provides you with the given actors orientation. You should only use this command if you have a special reason to do so, otherwise use the dark basic angle commands.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
outVec3ID - ID number of the vector3 the result will be placed into.


Returns:
None

DYN GET ORIENTATION ( int objectID, int limbID, int outVec3ID )

This command calculates and provides you with the given actors orientation. You should only use this command if you have a special reason to do so(such as retrieving bone angles), otherwise use the dark basic angle commands.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb which the actor represents in the simulation.
outVec3ID - ID number of the vector3 the result will be placed into.


Returns:
None
