DYN FLUID SCALE OBJECTS (int fluidID, int initialObjectID, float particleLife, float scaleStart, float scaleFinish)

This command will scale all your fluid objects, you must call this function in your game loop.

Parameters:
fluidID - ID number of the fluid.
initialObjectID - First object which was assigned to this fluid.
particleLife - Particle life you assigned to this fluid.
scaleStart - Scale the paricles will be when created. A value of 100 will mean the objects will not be scaled upon creation.
scaleFinish - Scale the particles will be when their life expires, A value of 100 will mean the objects will not be scaled as their life expires.


Returns:
None
