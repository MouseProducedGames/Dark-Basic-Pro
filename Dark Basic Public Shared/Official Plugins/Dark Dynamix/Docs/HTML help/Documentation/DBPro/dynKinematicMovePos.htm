DYN KINEMATIC MOVE POS (int objectID, float posX, float posY, float posZ)

If an actor has been made kinematic then you may use this command to position the actor.

Parameters:
objectID - ID number of the object/actor.
posX - Global position along the x axis.
posY - Global position along the y axis.
posZ - Global position along the z axis.


Returns:
None
