DYN JOINT SET DRIVE ORIENTATION (int jointID, float w, float x, float y, float z)

Set the drive goal orientation when it is being driven. The goal orientation is specified relative to the joint frame corresponding to actor[0].

Parameters:
jointID - ID number of the joint.
w - Quaternion element.
x - Quaternion element.
y - Quaternion element.
z - Quaternion element.


Returns:
None
