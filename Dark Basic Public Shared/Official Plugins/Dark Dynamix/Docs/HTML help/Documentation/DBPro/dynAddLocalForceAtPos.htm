DYN ADD LOCAL FORCE AT POS (int objectID, float x, float y, float z, float posX, float posY, float posZ)

Applies a force (or impulse) defined in the actor local coordinate frame, acting at a particular point in global coordinates, to the actor. You can change the force mode with the 'set force mode' command.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
x - Force to be applied along the x axis.
y - Force to be applied along the y axis.
z - Force to be applied along the z axis.
posX - Position the force is to be applied to.
posY - Position the force is to be applied to.
posZ - Position the force is to be applied to.


Returns:
None

DYN ADD LOCAL FORCE AT POS (int objectID, int limbID, float x, float y, float z, float posX, float posY, float posZ)

Applies a force (or impulse) defined in the actor local coordinate frame, acting at a particular point in global coordinates, to the actor. You can change the force mode with the 'set force mode' command. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
x - Force to be applied along the x axis.
y - Force to be applied along the y axis.
z - Force to be applied along the z axis.
posX - Position the force is to be applied to.
posY - Position the force is to be applied to.
posZ - Position the force is to be applied to.


Returns:
None
