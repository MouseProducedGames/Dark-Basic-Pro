int DYN CONTROLLER MOVE (int ID, float x, float y, float z)

This will move your character the given amount on each axis in global space. Return the collision flag which gives you information about current collisions around the controller.

Parameters:
objectID - ID number of the controller.
x - Distance to move along the x axis.
y - Distance to move along the y axis.
z - Distance to move along the z axis.


Returns:
Flag which gives you information on current contacts above, below, to the sides or a combination of the three.
