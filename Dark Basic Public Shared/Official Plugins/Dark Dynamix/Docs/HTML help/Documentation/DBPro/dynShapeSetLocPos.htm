DYN SHAPE SET LOC POS (int actorID, int shapeID, float posX, float posY, float posZ)

Sets the position of the shape relative to the local frame of the actor.

Parameters:
objectID ID number of the object/actor to which the shape belongs.
shapeID Index ID of the shape.
posX Position along the X axis.
posY Position along the Y axis.
posZ Position along the Z axis.


Returns:
None
