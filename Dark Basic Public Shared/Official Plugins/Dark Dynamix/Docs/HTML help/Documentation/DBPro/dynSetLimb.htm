DYN SET LIMB (int objectID, int limbID, bool physics)

Setting the second parameter to TRUE will give the physics control of the limbs in the given object. It also returns the limb back to animation control when the second parameter is set to FALSE.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb you wish to set.
physics - TRUE will give limb control to physics simulation, FALSE will release the limb from physics control.


Returns:
None
