int DYN MAKE SHAPE CONVEX (int objectID, int limbID, int convexMeshID)

Creates a shape from a convex mesh. The shape is added to a list, you can add as many shapes as you like to the list and then either create an actor from the shapes or add them to an existing actor.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the shape will position/orientate to.
convexMeshID - ID number of the convex mesh you wish to instance.


Returns:
The amount of shapes currently held in the list, if an actor is created from the current list of shapes then they will keep the shape number returned from this function.
