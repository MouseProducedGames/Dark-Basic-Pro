DYN FLUID FADE OBJECTS ALPHA (int fluidID, int initialObjectID, float particleLife, float fadeIn, float fadeOut)

This command will fade all your fluid objects, you must call this function in your game loop.

Parameters:
fluidID - ID number of the fluid.
initialObjectID - First object which was assigned to this fluid.
particleLife - Particle life you assigned to this fluid.
fadeIn - Amount of time in seconds it should take for the particles to fade in.
fadeOut - Amount of time in seconds it should take for the particles to fade out.


Returns:
None
