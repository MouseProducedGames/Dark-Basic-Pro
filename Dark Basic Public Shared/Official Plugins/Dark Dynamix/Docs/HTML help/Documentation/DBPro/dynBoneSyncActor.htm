DYN BONE SYNC ACTOR (int objectID, int meshID, int boneID)

This command will position and orientate the actor to the bone it represents in the simulation. The bones of your character model will change whenever the animation frame is changed or if the model is orientated/positioned in the world, you would call this command just before the physics takes over control of the bone.

Parameters:
objectID - ID number of the object.
meshID - ID number of the mesh which the bones belong.
boneID - ID number of the bone.


Returns:
None
