DYN JOINT SET DRIVE ANGULAR VELOCITY (int jointID, float x, float y, float z)

Set the drive angular velocity goal when it is being driven. The drive angular velocity is specified relative to the drive orientation target in the case of a slerp drive. The drive angular velocity is specified in the actor[0] or (ActorA) joint frame in all other cases.

Parameters:
jointID - ID number of the joint.
x - Velocity about the x axis.
y - Velocity about the y axis.
z - Velocity about the z axis.


Returns:
None
