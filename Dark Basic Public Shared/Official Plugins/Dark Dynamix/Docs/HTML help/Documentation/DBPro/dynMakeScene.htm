bool DYN MAKE SCENE (int sceneID)

Creates a new scene.

Parameters:
sceneID - ID number you wish to assign to the new scene.


Returns:
TRUE if succeeded, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN MAKE SCENE (int sceneID, bool hardwareScene)

Creates a new scene with the option of creating a hardware scene.

Parameters:
sceneID - ID number you wish to assign to the new scene.
hardwareScene - TRUE will attempt to create a hardware scene, FALSE will create a software scene.


Returns:
TRUE if succeeded, FALSE otherwise.
