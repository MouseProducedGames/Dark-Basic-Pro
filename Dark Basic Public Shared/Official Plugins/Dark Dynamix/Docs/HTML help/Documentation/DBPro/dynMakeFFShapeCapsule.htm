bool DYN MAKE FF SHAPE CAPSULE (int posVector3ID, int rotVector3ID, float radius, float height)

This command will add a calsule shape to an internal force field shape list, when you create a force field shape group it uses the shapes contained in this list, the list can then be cleared.

Parameters:
posVec3ID - ID number of the vector containing position data.
rotVec3ID - ID number of the vector containing rotation data.
radius - Radius for the capsule shape.
height - Height value for the capsule shape.


Returns:
TRUE if succeeded, FALSE otherwise.
