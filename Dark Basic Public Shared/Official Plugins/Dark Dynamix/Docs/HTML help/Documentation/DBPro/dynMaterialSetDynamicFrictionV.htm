DYN MATERIAL SET DYNAMIC FRICTION V (int ID, float value)

Sets the dynamic friction coefficient along the secondary (V) axis. This is used when anisotropic friction is being applied. I.e. the NX_MF_ANISOTROPIC flag is set.

Parameters:
ID - ID number of the material.
value - Coefficient of dynamic friction in the V axis. Range: [0, +inf]


Returns:
None
