DYN GET LINEAR MOMENTUM (int objectID, int vector3ID)

Fetches the linear momentum of an actor and places the values in the given vector.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
vector3ID - ID number of the vector you wish to use to collect the data.


Returns:
None


-------------------------------------------------------------------------------
DYN GET LINEAR MOMENTUM (int objectID, int limbID, int vec3Out)

Fetches the linear momentum of an actor. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb/bone the actor represents in the simulation.
vec3Out - ID number of the vector3 in which the data will be placed.


Returns:
None
