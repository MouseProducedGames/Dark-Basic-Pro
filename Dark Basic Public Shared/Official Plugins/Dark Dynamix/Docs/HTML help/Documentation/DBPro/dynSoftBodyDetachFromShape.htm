DYN SOFT BODY DETACH FROM SHAPE (int softBodyID, int objectID, int shapeIndex)

This command will detach the soft body from a rigid body shape.

Parameters:
softBodyID - ID number of the soft body.
objectID - ID number of the object whose associated actor is currently attached to the soft body.
shapeIndex - Index number of the shape you wish to detach, the default shape of an actor usually has an index number of 0.


Returns:
None

DYN SOFT BODY DETACH FROM SHAPE (int softBodyID, int objectID, int limbID, int shapeIndex)

This command will detach the soft body from a rigid body shape.

Parameters:
softBodyID - ID number of the soft body.
objectID - ID number of the limbs owning object.
limbID - ID number of the limb whose associated actor is currently attached to the soft body.
shapeIndex - Index number of the shape you wish to detach, the default shape of an actor usually has an index number of 0.


Returns:
None
