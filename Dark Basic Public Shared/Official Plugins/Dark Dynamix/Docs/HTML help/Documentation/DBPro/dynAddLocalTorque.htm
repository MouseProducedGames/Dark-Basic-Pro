DYN ADD LOCAL TORQUE (int objectID, float x, float y, float z)

Applies an impulsive torque defined in the local coordinate frame of the actor. You can change the force mode with the 'set force mode' command.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
x - Torque to be applied about the x axis.
y - Torque to be applied about the y axis.
z - Torque to be applied about the z axis.


Returns:
None


-------------------------------------------------------------------------------
DYN ADD LOCAL TORQUE (int objectID, int limbID, float x, float y, float z)

Applies an impulsive torque defined in the local coordinate frame of the actor. You can change the force mode with the 'set force mode' command. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
x - Torque to be applied about the x axis.
y - Torque to be applied about the y axis.
z - Torque to be applied about the z axis.


Returns:
None
