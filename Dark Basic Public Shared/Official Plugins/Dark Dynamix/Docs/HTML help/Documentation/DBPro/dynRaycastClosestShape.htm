bool DYN RAYCAST CLOSEST SHAPE (int origVec3ID, int dirVec3ID, int flag, int groups)

Casts a ray from a given position in the scene in a given direction. This can record a maximum of one hit, you can get the hit data using the 'raycast hit' commands.

Parameters:
origVec3ID - ID number of the vector containing the global position data.
dirVec3ID - ID number of the vector containing the global direction data, this vector must be normalized.
flag - NX_STATIC_SHAPES = 1, NX_DYNAMIX_SHAPES = 2, NX_ALL_SHAPES = 3
groups - Groups mask to filter shapes.


Returns:
TRUE if hit, FALSE otherwise.
