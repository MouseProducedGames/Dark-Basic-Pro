float DYN FLUID GET RANDOM ANGLE (int fluidID, int emitterIndex)

This command returns the maximum angle the particles leave the emitter, the default is for the particles to leave the emitter at the exact angle the emitter is facing, which is not always desirable.

Parameters:
fluidID - ID number of the fluid.
emitterIndex - Index of the emitter for the given fluid.


Returns:
- Angle value in radians.
