bool DYN OVERLAP CHECK AABB (int worldBoundsMinVec3ID, int worldBoundsMaxVec3ID, int flag)

Checks for any shapes within the given world bounds.

Parameters:
worldBoundsMinVec3ID ID number of the vector holding the minimum bounds.
worldBoundsMaxVec3ID ID number of the vector holding the maximum bounds.
flag Allows you to test for static, dynamic or all shapes(static - 1, dynamic - 2, all - 3).


Returns:
TRUE if a shape is detected within given bounds, FALSE otherwise.
