DYN UPDATE SOFT BODY DEBUG (int objectID, int softBodyID)

This command will set the soft body debug object to represent the current state of the soft body in the simulation, call this in your loop to keep it in sync with the simulation.

Parameters:
objectID - ID number of the soft body debug object.
softBodyID - ID number of the soft body which is represented by the terehedral mesh you wish to debug.


Returns:
None
