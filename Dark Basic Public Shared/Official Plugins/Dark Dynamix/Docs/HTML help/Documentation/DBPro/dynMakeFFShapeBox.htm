bool DYN MAKE FF SHAPE BOX (int posVector3ID, int rotVector3ID, int dimVector3ID)

This command will add a box shape to an internal force field shape list, when you create a force field shape group it uses the shapes contained in this list, the list can then be cleared.

Parameters:
posVec3ID - ID number of the vector containing position data.
rotVec3ID - ID number of the vector containing rotation data.
dimVec3ID - ID number of the vector containing dimension data.


Returns:
TRUE if succeeded, FALSE otherwise.
