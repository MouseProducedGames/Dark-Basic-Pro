DYN FLUID DELETE EMITTER (int fluidID, int emitterIndex)

This command will remove the given emitter from the given fluid.

Parameters:
fluidID - ID number of the fluid the given emitter belongs to.
emitterIndex - Index of the emitter, the index was returned when the emitter was created, if there is only one emitter for the given fluid then the index will be 0.


Returns:
None
