bool DYN MAKE FF LINEAR KERNEL (int ID, int constantVector3ID, int falloffLinearVector3ID, int falloffQuadraticVector3ID, int noiseVector3ID)

This command will create a linear kernel for use with a force field, this structure holds data which will determine how, and by how much, the force field will affect objects at different positions in the force field.

Parameters:
ID - ID number you wish to assign to this kernel.
constantVector3ID - The constant part of force field function. For a force field with cartesian coordinate space, this would affect the CORE direction objects would move in the force field.
falloffLinearVector3ID - Affects how strong the force field is relative to its centre with linear falloff.
falloffQuadraticVector3ID - Affects how strong the force field is relative to its centre with quadratic falloff.
noiseVector3ID - Adds the given amount of randomness to how objects are affected along each axis.


Returns:
