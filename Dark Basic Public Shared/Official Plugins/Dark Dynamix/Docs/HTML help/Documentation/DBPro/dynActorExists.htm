bool DYN ACTOR EXISTS (int objectID)

Checks an object to see whether it has an actor assigned to it.

Parameters:
objectID ID number of the object you wish to check.


Returns:
TRUE if actor exists, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN ACTOR EXISTS (int objectID, int limbID)

Checks an object to see whether it has an actor assigned to it.

Parameters:
objectID ID number of the object which the limb belongs.
limbID ID number of the limb you wish to check.


Returns:
TRUE if actor exists, FALSE otherwise.
