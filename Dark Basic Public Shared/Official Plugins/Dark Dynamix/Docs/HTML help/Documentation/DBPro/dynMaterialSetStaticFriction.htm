DYN MATERIAL SET STATIC FRICTION (int ID, float value)

Sets the coefficient of static friction. The coefficient of static friction should be in the range [0, +inf] if flags & NX_MF_ANISOTROPIC is set, then this value is used for the primary direction of anisotropy (U axis)

Parameters:
ID - ID number of the material.
value - Coefficient of static friction. Range: [0, +inf]


Returns:
None
