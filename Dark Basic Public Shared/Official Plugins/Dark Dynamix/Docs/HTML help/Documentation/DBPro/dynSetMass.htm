DYN SET MASS (int objectID, float mass)

Sets the mass of the given actor, mass must be positive and actor must be dynamic. This command does not update the inertial properties of the actor, to change the inertia tensor use the command that updates mass from shapes.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
mass - New mass of the given actor.


Returns:
None


-------------------------------------------------------------------------------
DYN SET MASS (int objectID, int limbID, float mass)

Sets the mass of the given actor, mass must be positive and actor must be dynamic. This command does not update the inertial properties of the actor, to change the inertia tensor use the command that updates mass from shapes. Use this overload if your actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
mass - New mass of the given actor.


Returns:
None
