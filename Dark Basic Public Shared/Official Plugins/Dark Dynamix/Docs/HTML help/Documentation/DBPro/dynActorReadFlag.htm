bool DYN ACTOR READ FLAG (int objectID, unsigned int flag)

Reads the given flag of the given actor.

Parameters:
objectID - ID number of the object the actor represents in the simulation.
flag - Flag you wish to read.


Returns:
TRUE if flag is raised, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN ACTOR READ FLAG (int objectID, int limbID, unsigned int flag)

Reads the given flag in the given actor. Use this overload if your actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
flag - Flag you wish to read.


Returns:
TRUE if flag is raised, false otherwise.
