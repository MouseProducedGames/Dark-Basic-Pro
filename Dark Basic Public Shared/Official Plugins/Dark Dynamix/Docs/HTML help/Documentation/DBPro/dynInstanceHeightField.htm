bool DYN INSTANCE HEIGHT FIELD (int ID, int heightFieldID, float heightScale, float rowScale, float columnScale)

After creating a height field descriptor, filling in the data and then creating a height field you can finally create your height field actor using this command.

Parameters:
ID - ID number of the object that the height field will represent. As with most static actors there is a very loose connection between actor and object, they share an ID number and not much else, you do need an object to exist though as it uses its position upon creation.
heightFieldID - ID number of the height field you wish to instance.
heightScale - Scale value for the height.
rowScale - Scale value for the rows.
columnScale - Scale value for the columns.


Returns:
TRUE if succeeded, FALSE otherwise.

bool DYN INSTANCE HEIGHT FIELD (int ID, int heightFieldID, float heightScale, float rowScale, float columnScale, int materialIndexHighBits, int holeMaterial)

After creating a height field descriptor, filling in the data and then creating a height field you can finally create your height field actor using this command. This overload allows to to set a materialIndexhighBits value and a hole material.

Parameters:
ID - ID number of the object that the height field will represent. As with most static actors there is a very loose connection between actor and object, they share an ID number and not much else, you do need an object to exist though as it uses its position upon creation.
heightFieldID - ID number of the height field you wish to instance.
heightScale - Scale value for the height.
rowScale - Scale value for the rows.
columnScale - Scale value for the columns.
materialIndexHighBits - The 9 high bits of this number are used to complete the material indices, the low 7 bits must be 0.
holeMaterial - The material index which designated holes in the height field.


Returns:
TRUE if succeeded, FALSE otherwise.
