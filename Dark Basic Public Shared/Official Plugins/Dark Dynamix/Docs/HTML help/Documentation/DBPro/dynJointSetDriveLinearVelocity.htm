DYN JOINT SET DRIVE LINEAR VELOCITY (int jointID, float x, float y, float z)

Set the drive goal linear velocity when it is being driven. The drive linear velocity is specified relative to the actor[0] joint frame.

Parameters:
jointID - ID number of the joint.
x - Velocity along the x axis.
y - Velocity along the y axis.
z - Velocity along the z axis.


Returns:
None
