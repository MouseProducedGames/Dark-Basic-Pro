DYN JOINT DESC GET LOCAL ANCHOR (int jointDescID, int AorB, int vector3Out)

Gets the local anchor point of one of the two actors associated with the joint descriptor.

Parameters:
jointDescID ID number of the descriptor.
AorB - 0 for actorA, 1 for actorB.
vector3Out - ID number of the vector3 the values will be placed.


Returns:
None
