DYN JOINT SET DRIVE POSITION (int jointID, float x, float y, float z)

Set the drive position goal position when it is being driven. The goal position is specified relative to the joint frame corresponding to actor[0].

Parameters:
jointID - ID number of the joint.
x - The goal position if NX_D6JOINT_DRIVE_POSITION is set for xDrive.
y - The goal position if NX_D6JOINT_DRIVE_POSITION is set for yDrive.
z - The goal position if NX_D6JOINT_DRIVE_POSITION is set for Drive.


Returns:
None
