DYN EMITTER SET FRAME SHAPE (int fluidID, int emitterIndex, int objectID, int shapeIndex)

This command will glue the emitter to a given shape of a given actor. The world position of the emitter will become the local position relative to the shape it has been attached to.

Parameters:
fluidID - ID number of the fluid the given emitter belongs to.
emitterIndex - Index of the emitter, the index was returned when the emitter was created, if there is only one emitter for the given fluid then the index will be 0.
objectID - ID number of the object whose actor you wish to glue the emitter to.
shapeIndex - Index of the shape inside the given actor, default shape is 0.


Returns:
None
