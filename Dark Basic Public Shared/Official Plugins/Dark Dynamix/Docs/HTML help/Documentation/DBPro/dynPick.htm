DYN PICK (int cameraID, int screenX, int screenY, int flag)

Helper function to give you quick control of your actors using the mouse pointer.

Parameters:
cameraID - ID number of the camera you are using.
screenX - You will usually put the x coordinate of your mouse pointer here.
screenY - You will usually put the y coordinate of your mouse pointer here.
flag - Defines what type of actors you wish to drag: none-0 static-1 dynamic-2 cloth-4 softbody-8 all-15


Returns:
None
