int DYN MAKE SHAPE TRIANGLE MESH (int objectID, int triangleMeshID)

Creates a shape from a convex mesh. The shape is added to a list, you can add as many shapes as you like to the list and then either create an actor from the shapes or add them to an existing actor. Do not add a triangle mesh shape to an actor that is dynamic as triangle meshes are static only.

Parameters:
objectID ID number of the object the shape will position/orientate to.
triangleMeshID ID number of the convex mesh you wish to instance.


Returns:
The amount of shapes currently held in the list, if an actor is created from the current list of shapes then they will keep the shape number returned from this function.

int DYN MAKE SHAPE TRIANGLE MESH (int objectID, int limbID, int triangleMeshID)

Creates a shape from a triangle mesh. The shape is added to a list, you can add as many shapes as you like to the list and then either create an actor from the shapes or add them to an existing actor. Do not add a triangle mesh shape to an actor that is dynamic as triangle meshes are static only.

Parameters:
objectID ID number of the owning object.
limbID ID number of the limb the shape will position/orientate to.
triangleMeshID ID number of the triangle mesh you wish to instance.


Returns:
The amount of shapes currently held in the list, if an actor is created from the current list of shapes then they will keep the shape number returned from this function.
