bool DYN MAKE CAPSULE (int objectID, float height, float radius, float density)

Creates a capsule actor.

Parameters:
objectID - ID number of the object the capsule will represent in the simulation.
height - The distance between the two hemispherical ends of the capsule.
radius - The radius of the capsules hemispherical ends and its trunk.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN MAKE CAPSULE (int objectID, int boneID, int meshID, float density)

Creates a capsule actor. This overload creates the capsule from a bone in the given mesh of the object.

Parameters:
objectID - ID number of the owning object.
boneID - ID number of the bone the actor will represent in the simulation.
meshID - ID number of the owning mesh.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.
