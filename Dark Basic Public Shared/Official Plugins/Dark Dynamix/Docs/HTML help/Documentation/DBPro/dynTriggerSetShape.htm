DYN TRIGGER SET SHAPE (int objectID, int shapeIndex, int flag, bool value)

This command allows you to turn an actors shape into a trigger, you can use the other trigger commands to retrieve trigger infromation when another shape/actor enters the trigger shape.

Parameters:
objectID ID number of the object which the actor represents in the simulation.
shapeIndex Index number of the shape you wish to become a trigger, default shape is 0.
isTrigger TRUE to set as trigger, FALSE otherwise.


Returns:
None
