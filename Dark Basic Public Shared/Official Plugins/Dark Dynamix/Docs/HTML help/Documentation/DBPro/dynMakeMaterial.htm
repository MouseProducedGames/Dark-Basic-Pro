bool DYN MAKE MATERIAL (int materialID, float restitution, float staticFriction, float dynamicFriction)

Creates a material which you can then apply to actor shapes, producing the effect of the actor being composed of a certain material.

Parameters:
materialID - ID number you wish to assign to the material.
restitution - The 'bounciness' of the material, a value between 0.0 and 1.0.
staticFriction - The friction of the material while the shape is static, a value between 0.0 and 1.0.
dynamicFriction - The friction of the material while the shape is moving, a value between 0.0 and 1.0.


Returns:
True if successful, FALSE otherwise.
