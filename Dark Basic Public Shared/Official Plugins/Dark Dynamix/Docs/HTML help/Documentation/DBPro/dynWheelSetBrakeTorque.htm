DYN WHEEL SET BRAKE TORQUE (int objectID, int shapeIndex, float torque)

Use this command to set the brake torque of the given wheel shape, this must be positive, high values will lock the wheel but should be stable.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
torque - Value you wish to set the brake torque to.


Returns:
None
