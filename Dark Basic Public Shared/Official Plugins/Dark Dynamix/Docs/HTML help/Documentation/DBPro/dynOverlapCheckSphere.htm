bool DYN OVERLAP CHECK SPHERE (int positionVec3ID, float radius, int flag)

Checks for any shapes within a sphere shape.

Parameters:
positionVec3ID ID number of the vector holding the position of the sphere.
radius Radius of the sphere.
flag Allows you to test for static, dynamic or all shapes(static - 1, dynamic - 2, all - 3).


Returns:
TRUE if a shape is detected in overlap, FALSE otherwise.
