bool DYN MAKE CONTROLLER BOX (int objectID, float yOffset)

Creates a character controller for the given object, character controllers are not controlled by the simulation but by the user.

Parameters:
objectID - ID number of the object you wish the controller to represent.
yOffset - Most character models have their local origin at their feet, use this command to raise the controller to the same position as your character. This value will usually be half the characters height.


Returns:
- TRUE if successful, FALSE otherwise.

bool DYN MAKE CONTROLLER BOX (int objectID, float yOffset, float slopeLimit, float stepOffset)

Creates a character controller for the given object, character controllers are not controlled by the simulation but by the user.

Parameters:
objectID - ID number of the object you wish the controller to represent.
yOffset - Most character models have their local origin at their feet, use this command to raise the controller to the same position as your character. This value will usually be half the characters height.
slopeLimit - Maximum slope, in degrees, that the controller will be able to climb.
stepOffset - Maximum step height that the controller will be able to climb.


Returns:
- TRUE if successful, FALSE otherwise.
