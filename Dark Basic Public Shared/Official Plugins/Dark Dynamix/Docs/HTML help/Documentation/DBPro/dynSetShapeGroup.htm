DYN SET SHAPE GROUP (int actorID, int shapeIndex, int group)

This command allows you to set which collision group the given shape belongs to, all shapes start off in group 0.

Parameters:
actorID - ID number of the object that the actor represents in the simulation.
shapeIndex - Index of the shape you wish to assign to a group. Set to -1 to assign all shapes in this actor to the group.
group - Group number you wish to assign the given shape to. Range: 0 - 31


Returns:
None

DYN SET SHAPE GROUP (int actorID, int limbID, int shapeIndex, int group)

This command allows you to set which collision group the given shape belongs to, all shapes start off in group 0.

Parameters:
actorID - ID number of the owning object.
limbID - ID number of the limb which the actor represents in the simulation.
shapeIndex - Index of the shape you wish to assign to a group. Set to -1 to assign all shapes in this actor to the group.
group - Group number you wish to assign the given shape to. Range: 0 - 31


Returns:
None
