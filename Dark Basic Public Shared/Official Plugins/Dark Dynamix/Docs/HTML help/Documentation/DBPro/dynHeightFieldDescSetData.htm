DYN HEIGHT FIELD DESC SET DATA (int heightFieldDescID, int row, int column, int height)

This command allows you to set the height data for the given height field descriptor.

Parameters:
ID - ID number of the height field descriptor.
column - Column index value.
row - Row index value.
height - Height value for the given location.


Returns:
None

DYN HEIGHT FIELD DESC SET DATA (int heightFieldDescID, int row, int column, int height, int materialIndex0, int materialIndex1, int tessFlag)

This command allows you to set the height data for the given height field descriptor. This overload allows you to set material indices and a tesselation flag.

Parameters:
ID - ID number of the height field descriptor.
column - Column index value.
row - Row index value.
height - Height value for the given location.
materialIndex0 - The low 7 bits of the triangle material index, these are concatenated with the high bits you can set when creating the actor to produce an index into the scenes material array. This value determines the value for the lower of the quads two triangles.
materialIndex1 - The low 7 bits of the triangle material index, these are concatenated with the high bits you can set when creating the actor to produce an index into the scenes material array. This value determines the value for the higher of the quads two triangles..
tessFlag - This flag specifies which way the quad is split.


Returns:
None
