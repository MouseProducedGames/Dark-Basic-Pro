bool DYN MAKE BOX (int objectID, float density)

Creates a box actor.

Parameters:
objectID - ID number of the object the box will represent in the simulation.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN MAKE BOX (int objectID, int boneID, int meshID, float density)

Creates a box actor. This overload creates the box from a bone in the given mesh of the object.

Parameters:
objectID - ID number of the owning object.
boneID - ID number of the bone the actor will represent in the simulation.
meshID - ID number of the owning mesh.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.
