DYN WHEEL SET SUSPENSION SPRING (int objectID, int shapeIndex, float spring, float damper, float targetValue)

Sets the spring variables on the given wheels suspension.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
spring - Spring coefficient.
damper - Damper coefficient, default value is 0.
targetValue - Target position of spring where the spring force is 0, default value is 0.


Returns:
None
