DYN SET ANGULAR MOMENTUM (int objectID, float x, float y, float z)

Sets the angular momentum of an actor, you should usually apply forces to change the velocity of an actor although this command can be used to reset your actor etc.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
x - Momentum about the x axis.
y - Momentum about the y axis.
z - Momentum about the z axis.


Returns:
None


-------------------------------------------------------------------------------
DYN SET ANGULAR MOMENTUM (int objectID, int limbID, float x, float y, float z)

Sets the angular momentum of an actor. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb/bone the actor represents in the simulation.
x - Angular damping value about the x axis.
y - Angular damping value about the y axis.
z - Angular damping value about the z axis.


Returns:
None
