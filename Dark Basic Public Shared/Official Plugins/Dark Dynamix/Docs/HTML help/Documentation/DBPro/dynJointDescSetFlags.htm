DYN JOINT DESC SET FLAGS (int jointDescID, int flags)

The flag value determines whether debug visualisation of the joint will be enabled(default) and also whether the joined actors will collide with each other(default = no collision).

Parameters:
jointDescID - ID number of the descriptor.
flags - Combination of values to determine visualisation(NX_JF_VISUALIZATION = 2) and collision(NX_JF_COLLISION_ENABLED = 1).


Returns:
None
