float DYN WHEEL GET SUSPENSION TRAVEL (int objectID, int shapeIndex)

This command will return the maximum travel distance along the wheel shapes -Y axis, the minimum suspension travel is always 0.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.


Returns:
Maximum suspension travel distance for the given wheel shape.
