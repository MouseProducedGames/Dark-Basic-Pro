bool DYN MAKE WHEEL (int objectID, float density)

Creates a Wheel actor.

Parameters:
objectID - ID number of the object the wheel will represent in the simulation.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.


-------------------------------------------------------------------------------
bool DYN MAKE WHEEL (int objectID, int limbID, float radius, float density)

Creates a Wheel actor from a limb.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the wheel will represent in the simulation.
density - The density of the actor, along with its size this will determine how heavy the actor is.


Returns:
TRUE if succeeded, FALSE otherwise.
