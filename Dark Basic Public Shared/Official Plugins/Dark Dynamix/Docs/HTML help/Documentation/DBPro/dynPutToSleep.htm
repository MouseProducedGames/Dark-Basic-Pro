DYN PUT TO SLEEP (int objectID)

Puts the given actor to sleep, actors fall asleep themselves when they are left idle long enough, saving cpu. Use this command if you need them to fall asleep in situations when they would not otherwise do so themselves.

Parameters:
objectID - ID number of object the actor represents in the simulation.


Returns:
None


-------------------------------------------------------------------------------
DYN PUT TO SLEEP (int objectID, int limbID)

Puts the given actor to sleep, actors fall asleep themselves when they are left idle long enough, saving cpu. Use this command if you need them to fall asleep in situations when they would not otherwise do so themselves. Use this overload if the actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of limb the actor represents in the simulation.


Returns:
None
