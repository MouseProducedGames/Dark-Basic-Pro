DYN SOFT BODY ADD FORCE AT POS (int softBodyID, int posVec3ID, float magnitude, float radius)

Applies a radial force (or impulse) at a particular position. All vertices within radius will be affected with a quadratic drop-off. Because forces are reset at the end of every timestep, you can maintain a total external force on an object by calling this once every frame.

Parameters:
softBodyID - ID number of the object the soft body represents in the simulation.
posVec3ID - ID number of the vector containing the position.
magnitude - Magnitude of the force/impulse to apply.
radius - The sphere radius in which particles will be affected.


Returns:
None
