DYN JOINT SET DRIVE ORIENTATION FROM GLOBAL AXIS (int jointID, float x, float y, float z)

Set the drive goal orientation when it is being driven. The goal orientation is specified relative to the joint frame corresponding to actor[0].

Parameters:
jointID - ID number of the joint.
x - Angle value about the x axis.
y - Angle value about the y axis.
z - Angle value about the z axis.


Returns:
None
