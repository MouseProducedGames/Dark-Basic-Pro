DYN FLUID SET OBJECTS TO CAMERA (int fluidID, int initialObjectID, int camID)

This command will point all your fluid objects toward the given camera, if you are using multiple cameras you would need to sync the first camera and then call this function again to point toward the second camera before syncing the second camera.

Parameters:
fluidID - ID number of the fluid.
initialObjectID - First object which was assigned to this fluid.
camID - ID number of the camera you wish to point the objects toward.


Returns:
None
