DYN GROUP SET COLLISION (int groupA, int groupB, bool enable)

This command allows you to set whether the actors in the given two shape groups collide with each other, all shapes start out in group 0 and all groups are enabled by default.

Parameters:
groupA - The first group of the pair you wish to set collision for.
groupB - the second group of the pair you wish to set collisions for.
enable - TRUE enables collisions, FALSE disables collisions.


Returns:
None
