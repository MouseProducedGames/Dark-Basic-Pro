bool DYN OVERLAP CHECK CAPSULE (int pos1Vec3ID, int pos2Vec3ID, float radius, int flag)

Checks for any shapes within the given capsule shaped bounds.

Parameters:
pos1Vec3ID ID number of the first position vector for one end of the capsule.
pos2Vec3ID ID number of the second position vector for the other end of the capsule.
flag Allows you to test for static, dynamic or all shapes(static - 1, dynamic - 2, all - 3).


Returns:
TRUE if a shape is detected within given bounds, FALSE otherwise.
