bool DYN OVERLAP CHECK OBB (int positionVec3ID, int dimensionVec3ID, int rotationVec3ID, int flag)

Checks for any shapes within the given orientated bounding box bounds.

Parameters:
positionVec3ID ID number of the position vector.
positionVec3ID ID number of the dimension vector.
dimensionVec3ID ID number of the rotation vector.
flag Allows you to test for static, dynamic or all shapes(static - 1, dynamic - 2, all - 3).


Returns:
TRUE if a shape is detected within given bounds, FALSE otherwise.
