DYN WHEEL SET SUSPENSION TRAVEL (int objectID, int shapeIndex, float suspensionTravel)

Sets the maximum extension distance along shapes local y-axis, minimum extension is 0;

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
suspensionTravel - Value you wish to set the suspension travel to.


Returns:
None
