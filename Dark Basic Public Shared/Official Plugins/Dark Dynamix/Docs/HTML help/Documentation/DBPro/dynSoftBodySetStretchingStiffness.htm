DYN SOFT BODY SET STRETCHING STIFFNESS (int softBodyID, float value)

Sets the stretching stiffness of the given soft body. Volume and stretching stiffnesses are used the specify how strongly the soft body responds to deviations from the rest state given by the input mesh. The soft body simulation manipulates the vertex positions to counteract deformations that change the volume and edge lengths of each tetrahedral element.

Parameters:
softBodyID - ID number of the soft body.
value - Value you wish to set the stretching stiffness, between 0.0 - 1.0, the default value is 0.5.


Returns:
None
