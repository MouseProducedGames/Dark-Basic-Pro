int DYN RAYCAST ALL SHAPES (int origVec3ID, int dirVec3ID)

Casts a ray from a given position in the scene in a given direction. You can then use the other raycast commands to retrieve any hit data. This command can generate more than one hit per raycast.

Parameters:
origVec3ID - ID number of the vector containing the global position data.
dirVec3ID - ID number of the vector containing the global direction data.


Returns:
Number of hits.

int DYN RAYCAST ALL SHAPES (int origVec3ID, int dirVec3ID, int flag, int groups)

Casts a ray from a given position in the scene in a given direction. You can then use the other raycast commands to retrieve any hit data. This command can generate more than one hit per raycast.

Parameters:
origVec3ID - ID number of the vector containing the global position data.
dirVec3ID - ID number of the vector containing the global direction data.
flag - NX_STATIC_SHAPES = 1, NX_DYNAMIX_SHAPES = 2, NX_ALL_SHAPES = 3
groups - Groups mask to filter shapes.


Returns:
Number of hits.
