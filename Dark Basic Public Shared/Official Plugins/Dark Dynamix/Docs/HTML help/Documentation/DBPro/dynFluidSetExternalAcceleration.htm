DYN FLUID SET EXTERNAL ACCELERATION (int fluidID, float x, float y, float z)

Sets the external acceleration of the given fluid. External acceleration is applied directly to particles. It can be used to counteract the effect of gravity or simulate wind, etc.

Parameters:
fluidID - ID number of the fluid.
x - Acceleration value along the global x axis.
y - Acceleration value along the global y axis.
z - Acceleration value along the global z axis.


Returns:
None
