DYN BONE GET POS (int objectID, int meshID, int boneID, int vec3Out)

Fetches the global position of the biven bone, this is useful when creating joints between bones for ragdolls. Most of the time you will find the bone position is the exact position you need to create the joint.

Parameters:
objectID - ID number of the object.
meshID - ID number of the mesh which the bones belong.
boneID - ID number of the bone.
vec3Out - ID number of the vector3 in which the positional data will be placed.


Returns:
None
