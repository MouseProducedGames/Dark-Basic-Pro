DYN MATERIAL SET DIR OF ANISOTROPY (int ID, float x, float y, float z)

Sets the shape space direction (unit vector) of anisotropy. This is used when anisotropic friction is being applied. I.e. the NX_MF_ANISOTROPIC flag is set.

Parameters:
ID - ID number of the material.
x - Direction along the x axis.
y - Direction along the y axis.
z - Direction along the z axis.


Returns:
None
