DYN SET LINEAR VELOCITY (int objectID, float x, float y, float z)

Sets the linear velocity of an actor, you should usually apply forces to change the velocity of an actor although this command can be used to reset your actor etc.

Parameters:
objectID - ID number of the object whose associated actor you wish to access.
x - Velocity along the x axis.
y - Velocity along the y axis.
z - Velocity along the z axis.


Returns:
None

DYN SET LINEAR VELOCITY (int objectID, int limbID, float x, float y, float z)

Sets the linear velocity of an actor, you should usually apply forces to change the velocity of an actor although this command can be used to reset your actor etc.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.
x - Velocity along the x axis.
y - Velocity along the y axis.
z - Velocity along the z axis.


Returns:
None
