DYN WHEEL SET LONG TIRE FORCE FUNC (int objectID, int shapeIndex, int tireFuncDescID)

Use this command to set the longitudal tire force function of the given wheel shape.

Parameters:
objectID - ID number of the object the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
tireFuncDesc - ID number of the tire function descriptor.


Returns:
None

DYN WHEEL SET LONG TIRE FORCE FUNC (int objectID, int limbID, int shapeIndex, int tireFuncDescID)

Use this command to set the longitudal tire force function of the given wheel shape.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the wheel represents in the simulation.
shapeIndex - Index value of the wheel shape in the actor, the default shape is 0.
tireFuncDesc - ID number of the tire function descriptor.


Returns:
None
