float DYN GET MASS (int objectID)

Returns the mass of the given actor, returns 0 for static actors.

Parameters:
objectID - ID number of the object the actor represents in the simulation.


Returns:
The mass of the given actor.


-------------------------------------------------------------------------------
float DYN GET MASS (int objectID, int limbID)

Returns the mass of the given actor, returns 0 for static actors, use this overload if your actor was created from a limb/bone.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb/bone the actor represents in the simulation.


Returns:
The mass of the given actor.
