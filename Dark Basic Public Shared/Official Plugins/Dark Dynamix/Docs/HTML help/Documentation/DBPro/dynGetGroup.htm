int DYN GET GROUP (int actorID)

This command allows you to retrieve which collision group the given actor belongs to, all actors start off in group 0.

Parameters:
actorID - ID number of the object that the actor represents in the simulation.


Returns:
Group number in which the given actor belongs.


-------------------------------------------------------------------------------
int DYN GET GROUP (int objectID, int limbID)

This command allows you to retrieve which collision group the given actor belongs to, all actors start off in group 0.

Parameters:
objectID - ID number of the owning object.
limbID - ID number of the limb the actor represents in the simulation.


Returns:
Group number in which the given actor belongs.
