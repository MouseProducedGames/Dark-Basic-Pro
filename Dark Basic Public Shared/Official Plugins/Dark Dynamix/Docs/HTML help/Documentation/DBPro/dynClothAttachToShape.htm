DYN CLOTH ATTACH TO SHAPE (int clothID, int objectID, int shapeIndex, unsigned int attachmentFlags)

Attaches the cloth to the given shape, it will attach the cloth vertices which are currently inside the given shape.

Parameters:
clothID - ID number of the object the cloth represents in the simulation.
objectID - ID number of the object the actor represents in the simulation.
shapeIndex - Shape index, default shape index of an actor is 0.
attachmentFlags - Flag or combination of flags which will determine whether the attachment is one way or two way or whether the attachment is tearable.


Returns:
None
