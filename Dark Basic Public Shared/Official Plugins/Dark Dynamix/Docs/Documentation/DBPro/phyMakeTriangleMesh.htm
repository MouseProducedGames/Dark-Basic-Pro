bool PHY MAKE TRIANGLE MESH (int objectID, int triangleMeshID)

Creates an actor from a triangle mesh, all triangle meshes are static.

Parameters:
objectID ID number of the object the actor will represent in the simulation.
triangleMeshID ID number of the mesh you wish to create the actor from.


Returns:
TRUE if succeeded, FALSE otherwise.
