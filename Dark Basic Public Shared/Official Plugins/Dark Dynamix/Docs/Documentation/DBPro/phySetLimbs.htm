PHY SET LIMBS (int objectID, bool physics)

Setting the second parameter to TRUE will give the physics control of the limbs in the given object. It also returns the limbs back to animation control when the second parameter is set to FALSE.

Parameters:
objectID ID number of the object which has a ragdoll set up.
physics TRUE will give limb control to physics simulation, FALSE will release the limbs from physics control.


Returns:
None
