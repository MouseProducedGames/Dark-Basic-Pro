PHY SET SHAPE GLOB ROT (int actorID, int shapeID, float angleX, float angleY, float angleZ)

Sets the rotation of the shape in global space.

Parameters:
objectID ID number of the object/actor to which the shape belongs.
shapeID Index ID of the shape.
angleX Rotation about the X axis.
angleY Rotation about the Y axis.
angleZ Rotation about the Z axis.


Returns:
None
