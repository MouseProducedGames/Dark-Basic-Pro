PHY SET JOINT SWING1 MOTION (int jointDescID, int motion)

Sets the degree of freedom for the swing1 motion of the given joint descriptor. If locked, any joint created using this descriptor will not allow rotation about the local y-axis.

Parameters:
jointDescID ID number of the descriptor.
motion Flag used to set degree of freedom, set to 0 for complete freedom(default), 1 for limited freedom, 2 for no freedom(locked).


Returns:
None
