bool PHY MAKE CONVEX MESH (int objectID, int convexMeshID, float density)

Creates an actor from a convex mesh.

Parameters:
objectID ID number of the object the actor will represent in the simulation.
convexMeshID ID number of the mesh you wish to create the actor from.
density Density value of dynamic actor, 0.0 for static actor.


Returns:
TRUE if succeeded, FALSE otherwise.
